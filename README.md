Git LFS is GitHub's solution for handling large files by storing them outside the Git repository while keeping lightweight references in the repo.

Free storage limit: Git LFS gives you 1 GB free storage and 1 GB monthly bandwidth per account

File size: The CSV will render as a table on GitHub only if it's under 512 KB. If larger, it will still be stored and downloadable but displayed as raw text

Once set up correctly, the CSV file will appear in the GitHub repository and be downloadable by anyone who clones it. Personal data must not be included.
